 Analyst Pro Tips

  • Top Content: In Sheets, go to Insert > Pivot Table. Set 'Title' as Rows and 'COUNTA of Title' as Values to see the most used screenshots.
  • User Count: Select the 'User Device ID' column and use the "Count Unique" function to see the actual number of agents.
  • Export Report: Go to File > Download > PDF to generate a clean report for weekly presentations.
